README — Instalação e Execução do Cypress
1. Instalar Node.js

Baixar e instalar:
https://nodejs.org/

Verificar instalação:

node -v
npm -v

2. Clonar o repositório
git clone https://github.com/SEU-USUARIO/SEU-REPO.git


Entrar no projeto:

cd SEU-REPOSITORIO

3. Instalar dependências
npm install

4. Abrir o Cypress (modo interativo)
npx cypress open

5. Executar os testes (modo headless)
npx cypress run

6. Estrutura gerada automaticamente

Screenshots:

cypress/screenshots/


Vídeos:

cypress/videos/

7. Instalar Cypress manualmente (opcional)
npm install cypress --save-dev

8. Verificar versão do Cypress
npx cypress -v
